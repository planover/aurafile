光匣 Aurafile

梦幻柔和的离线 NAS 文件管理器

专为飞牛 NAS（fnOS）fpk 窗口打造，所有数据始终留在你自己的设备里。

🔒 完全离线运行 · 无任何联网请求、外部 API、遥测与广告

✨ 核心特性

  • 完整文件操作：重命名 / 复制 / 剪切 / 撤销重做 / 回收站
  • 全文检索：按名称、类型、大小、时间，甚至文件内部文本
  • 缩略图与元信息：图片、视频抽帧预览，EXIF 查看
  • 压缩解压：zip / tar.gz / 7z，防 Zip-Slip 与解压炸弹
  • 增量索引与时间轴：实时监控、搜索秒开，按今天 / 本周 / 本月分组
